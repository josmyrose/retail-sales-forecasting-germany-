\## Data Source



The data used in this project is published by the Deutsche Bundesbank.



\- Dataset: Monthly Retail Trade Turnover Index (Germany)

\- Frequency: Monthly

\- Source: Deutsche Bundesbank



Due to licensing and redistribution considerations, the raw dataset is not stored in this repository.



To reproduce the analysis:

1\. Download the dataset directly from the Deutsche Bundesbank data portal

2\. Save the CSV file in the local project directory

3\. Update the file path in the notebooks if necessary
